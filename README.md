## Numerical Simulations  

For the simulations we use the Plank 2018 cosmological parameters:

$$H_0 = 667.66$$

$$\Omega_m = 0.3111$$

$$\Omega_\Lambda = 0.6889$$

$$\sigma_8 = 0.8102$$

$$n_s = 0.9665$$
